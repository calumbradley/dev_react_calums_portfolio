# Description
This project forks from [https://github.com/tbakerx/react-resume-template](https://github.com/tbakerx/react-resume-template)

# Run Project
### 1. Clone the project

```shell
git clone https://github.com/calumbradley/calums_portfolio.git
```

### 2. Run the project
```shell
npm i
npm start
```

### 3. Build
```shell
npm run build
```
